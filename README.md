# fxTap

## 简介

这是一款运行在卡西欧 fx-9750 与 fx-9860 系列图形计算器上的下落式音游。

本项目将抛弃原有的 Malody 铺面，转而使用 osu! 的铺面，因为 Malody 并非一款开源游戏，因此需要对其进行连蒙带猜的反向工程，非常不友好。而 osu! 则将自己铺面的格式公之于众，方便开发。该转换器仅仅识别铺面的一个子集，因为有很多功能在计算器上不需要，比如背景图像和音频。目前只考虑 osu!mania 的铺面。

这一次并没有使用卡西欧官方发布的远古 SDK，而是使用 Lephenixnoir 开发的 fxSDK 与 gint 内核。用了这个之后，绘图速度与计时器精度都得到了极其显著的提升。
